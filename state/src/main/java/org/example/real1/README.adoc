售后工单

. _工单新建_： -> 工单状态：*待领取*
. _客服领取_：*待领取* -> 工单状态：*客服处理中*；更新工单客服处理人
. _客服提交_：*客服处理中* -> 工单状态：工单信息 ? *待完结* : *财务处理中*
. _财务领取_：更新工单财务处理人
. _财务提交_：*财务处理中* -> 工单状态：*待完结*
. _财务退回_：*财务处理中* -> 工单状态：*客服处理中*
. _工单完结_：*待完结* -> 工单状态：*已完结*
. _工单作废_：*财务处理中* || *客服处理中* || *待领取* -> 工单状态：*已作废*






== States

初始状态：待领取
中间状态：客服处理中、财务处理中、待完结
终结状态：已完结、已作废

== Actions

* 工单新建
* 客服领取
* 客服提交
* 财务领取
* 财务提交
* 财务退回
* 工单完结
* 工单作废
